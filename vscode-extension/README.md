# HTML 可视化编辑器 - VS Code 插件

在 VS Code 中可视化编辑 HTML 文件，无需直接编辑代码。

## 功能特性

- 📄 **可视化编辑**：在预览界面直接点击元素进行编辑
- 🎨 **属性面板**：修改文字、颜色、字体、尺寸等属性
- 🔄 **实时预览**：所见即所得的编辑体验
- 💾 **自动保存**：编辑内容自动同步到文件
- 🎯 **零依赖**：无需额外配置，开箱即用

## 使用方法

### 方式 1：右键菜单
1. 在资源管理器中右键点击 `.html` 文件
2. 选择 "用可视化编辑器打开"

### 方式 2：编辑器标题栏
1. 打开任意 `.html` 文件
2. 点击编辑器标题栏的 "用可视化编辑器打开" 按钮

### 方式 3：命令面板
1. 按 `Cmd+Shift+P` (Mac) 或 `Ctrl+Shift+P` (Windows/Linux)
2. 输入 "HTML Editor: 用可视化编辑器打开"

## 编辑功能

### 选择元素
- 在预览区域点击任意元素即可选中
- 选中后右侧属性面板会显示该元素的属性

### 编辑文字
- **方式 1**：在属性面板的"文字内容"区域修改
- **方式 2**：在预览区域双击文字直接编辑

### 修改样式
在属性面板中可以修改：
- 文字大小和粗细
- 文字对齐方式
- 文字颜色和背景颜色
- 内边距和圆角
- 宽度和高度

### 特殊元素
- **图片 (`<img>`)**：可以修改图片地址 (src)
- **链接 (`<a>`)**：可以修改链接地址 (href)

### 删除元素
选中元素后，点击属性面板底部的"删除此元素"按钮

## 系统要求

- Visual Studio Code 1.75.0 或更高版本

## 已知限制

- 不支持编辑结构性标签（如 `<html>`, `<head>`, `<script>` 等）
- 仅支持内联样式编辑
