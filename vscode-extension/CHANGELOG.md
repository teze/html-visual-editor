# 更新日志

## [0.1.0] - 2026-05-30

### 新增
- 🎉 首次发布
- ✨ 可视化编辑 HTML 文件
- 🎨 属性面板支持修改文字、颜色、字体、尺寸等
- 🔄 实时预览和自动保存
- 📝 支持双击文字直接编辑
- 🗑️ 支持删除元素
- 🖼️ 支持编辑图片地址
- 🔗 支持编辑链接地址

### 特性
- 深度集成 VS Code
- 自动保存到文件
- 支持外部修改检测
- 右键菜单快速打开
- 零配置，开箱即用

### 已知限制
- 不支持编辑结构性标签（`<html>`, `<head>`, `<script>` 等）
- 仅支持内联样式编辑
- 不支持外部 CSS 文件编辑
