### 1. 元素树面板（Components Tree）

**功能：**
- ✅ 显示完整的 DOM 树结构
- ✅ 树节点可折叠/展开

## 🔄 交互流程

### 选中元素

1. **方式 1：点击树节点**
   - 在左侧 Components 树中点击节点
   - 预览中对应元素被选中
   - 右侧属性面板显示该元素的属性

2. **方式 2：点击预览**
   - 在预览区域点击元素
   - 左侧树中对应节点高亮
   - 右侧属性面板显示该元素的属性

### 编辑属性

1. 在右侧属性面板修改属性
2. 预览中实时更新
3. 左侧树保持选中状态

### 撤销/重做

- 点击"↶ 撤销"或按 `Ctrl+Z`
- 点击"↷ 重做"或按 `Ctrl+Y`
- 支持多次撤销/重做

## 🔧 技术细节

### 元素路径表示

使用数组表示元素在 DOM 树中的位置：
```javascript
// [0, 1, 2] 表示 body.children[0].children[1].children[2]
elementPath: [0, 1, 2]
```

### 树节点 ID

```javascript
// 路径 [0, 1, 2] 生成 ID "node-0-1-2"
nodeId: "node-0-1-2"
```

### 数据流

```
用户点击树节点
  ↓
onNodeSelect(path)
  ↓
找到对应元素
  ↓
element.click()
  ↓
iframe 触发 onClick
  ↓
post({ type: 'selected', info })
  ↓
MessageBus.onSelected(info)
  ↓
更新属性面板 + 更新树选中状态
```
