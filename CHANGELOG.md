# 更新日志

## [1.2.0] - 2026-05-31

### 重大改进

#### 撤销/重做重构：快照模式 → 命令模式
- 🔄 **完全重写撤销/重做机制**，从快照模式改为命令模式（Command Pattern）
- ✅ **修复关键 bug**：现在支持多次撤销/重做（之前只能操作一次）
- ⚡ **性能提升**：不再重新渲染整个 iframe，只修改具体元素
- 💾 **内存优化**：只保存操作命令，而非完整 HTML 快照
- 🎯 **更可靠**：不会触发额外事件，历史栈保持干净

#### 命令类型
- `UpdateStyle` - 样式更新命令（颜色、字体、尺寸等）
- `UpdateText` - 文字更新命令
- `UpdateAttribute` - 属性更新命令（src、href 等）
- `DeleteElement` - 删除元素命令（可恢复）

### 技术改进

#### 新增模块
- `src/command-history.js` - 基于命令模式的历史管理器
- 替代原有的 `src/history-manager.js`（快照模式）

#### 核心实现
- iframe 中的 `onMessage` 在应用更改前捕获旧值
- 创建命令对象并通过 `postMessage` 发送回父窗口
- 父窗口的 `commandHistory` 管理命令栈
- 撤销时执行反向操作，重做时执行正向操作
- 使用元素路径（索引数组）定位元素

#### 代码简化
- ❌ 删除 `isRestoring` 标志和相关逻辑
- ❌ 删除 `bus.detach()` / `attach()` 调用
- ❌ 删除 `historyManager.disable()` / `enable()` 调用
- ❌ 删除 `saveHistorySnapshot` 函数
- ✅ 代码更简洁，逻辑更清晰

### 自动保存重新启用
- 💾 命令模式不会触发 iframe 重新加载，因此可以安全地启用自动保存
- 🔄 自动保存和撤销/重做可以同时工作，互不干扰

### 文档更新
- 📖 新增 `COMMAND_PATTERN_IMPLEMENTATION.md` - 命令模式实现详解
- 📝 更新 `README.md` - 说明命令模式的优势
- 🧪 新增 `test-command-undo.html` - 命令模式测试页面

### 已知限制
- 元素路径基于子元素索引，如果 DOM 结构发生变化，路径可能失效
- `restoreElement` 中使用 `innerHTML`，存在潜在的 XSS 风险（但恢复的是用户自己的内容）

## [1.1.0] - 2026-05-31

### 新增功能

#### 撤销/重做（快照模式 - 已废弃）
- ✨ 添加撤销/重做功能，支持最多 50 步历史记录
- 🎯 顶栏新增"↶ 撤销"和"↷ 重做"按钮
- ⌨️ 支持键盘快捷键：`Ctrl+Z` 撤销，`Ctrl+Y` 重做（Mac: `Cmd+Z` / `Cmd+Y`）
- 🔘 按钮在无可用历史时自动禁用
- 📸 每次修改后自动保存历史快照
- 🔄 撤销/重做时完整恢复页面状态
- ⚠️ **已知问题**：只能撤销/重做一次（已在 v1.2.0 中修复）

#### 自动保存
- 💾 修改后 2 秒自动保存到浏览器本地存储（localStorage）
- 📊 顶栏显示保存状态："已保存 刚刚" / "已保存 X 分钟前"
- 🔔 刷新页面时自动检测并提示恢复未保存的内容
- 🗑️ 导出文件后自动清除保存记录
- ⚡ 防抖保存：避免频繁写入，提升性能

### 技术改进

#### 新增模块
- `src/history-manager.js` - 历史记录管理模块
- `src/auto-saver.js` - 自动保存管理模块

#### 核心功能
- 历史快照包含完整的 HTML 和选中状态
- 使用栈结构管理历史记录（最多 50 条）
- 自动保存使用 localStorage，支持跨会话恢复
- 保存元数据包括文件名、保存时间、文件大小

### 用户体验优化
- 🎨 撤销/重做按钮禁用时显示半透明样式
- 📍 保存状态实时更新，颜色区分不同状态
- 🚫 在输入框中输入时禁用键盘快捷键，避免冲突
- ⏱️ 加载文件时自动清空历史记录并保存初始快照

### 已知限制
- 历史记录仅在当前会话有效，刷新页面后清空
- 自动保存仅保存最新版本，不支持多版本历史
- localStorage 有大小限制（通常 5-10MB），超大文件可能保存失败

## [1.0.0] - 2026-05-29

### 初始版本

#### 核心功能
- 📄 可视化编辑 HTML 文件
- 🎨 属性面板修改样式
- 🔄 实时预览
- 💾 导出 HTML
- 🎯 零依赖单文件

#### 编辑功能
- 选择元素（悬停高亮、点击选中）
- 双击文字直接编辑
- 修改文字内容、字体大小、粗细、颜色
- 修改背景颜色、内边距、圆角、宽高
- 文字对齐（左/中/右）
- 图片 src 和链接 href 编辑
- 删除元素

#### 技术特性
- 单文件 HTML，可通过 file:// 直接打开
- iframe sandbox 隔离
- postMessage 通信
- 模块化开发，组装为单文件
